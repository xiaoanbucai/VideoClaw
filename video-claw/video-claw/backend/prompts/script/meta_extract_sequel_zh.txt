你是一位经验丰富的影视统筹助理。你的任务是从续写的连贯新剧本内容中，分析并提取**全新出现**的角色和场景。

---

【输入】

【原角色清单】（如果提取的人物和现有清单重复，请**不要**将其加入新人物列表）：
{existing_chars}

【原场景清单】（如果是已经存在或者能轻易并入已知场景的，**不要**加入新场景列表中）：
{existing_settings}

【新续写剧本内容】：
{sequel_script}

---

【设计要求】

1. **输出格式**：必须且仅能输出一个纯净的 JSON 对象。严禁包含任何 Markdown 格式（如 ```json 代码块）、解释性文字、开场白或结尾。
2. **提取对象**：提取新剧本中出现的、有台词或重要动作的【新角色】和【新场景】。注意忽略群体角色。
3. **角色描述**（用于生成静态肖像）：
   - 采用“年龄+性别+发型/首饰+服装+体型+面部特征”的概括性写法。
   - 包含发型、体型、服装颜色和款式等视觉特征。
   - 严格限制：绝不能涉及神态、表情、性格，只描写纯粹的客观外貌；不要描写人物的任何动作；必须是一个固定的、静态的形象描述；人物形象要保持整洁，衣服完好，不要描述伤口、血迹、衣物破损等；人物身上不要有任何特殊的标记（如疤痕、胎记、纹身、奇怪的面妆等）。
   - 风格要求：30-50字，语言平实直接，禁止使用艺术化修辞。
4. **场地布置**：
   - 采用“空间类型+光效方向与颜色+核心陈设（质感/材质）+环境氛围”的概括性写法。
   - **严格限制**：环境必须保持干净、整洁。设计必须避开所有脏乱差的环境。场地可以非常朴素简陋，但绝对不能脏乱、恶心、血腥或破烂不堪。
   - 风格要求：30-50字，描述具体且具备可拍摄感。

---

【输出要求】

请你判断基于【原角色清单】【原场景清单】，【新续写剧本内容】中是否存在新的主要角色和新出现的核心地点场景。如果存在，请提取出来，并按照严格的 JSON 结构返回。
如果有，请存入对应列表。如果没有新人物或新场景，则对应列表为空。
请直接输出 JSON 格式（不要使用 markdown 的 ``` 包裹，并且不要含有额外文字）。

返回的 JSON 的格式必须完全匹配：

{{
  "new_characters": [
    {{
      "name": "新角色全名",
      "description": "外貌描写，要求参见【设计要求】",
      "role": "main/supporting/minor中的一种分类"
    }}
  ],
  "new_settings": [
    {{
      "name": "新地点名称",
      "description": "概括性场地描述，要求参见【设计要求】"
    }}
  ]
}}
